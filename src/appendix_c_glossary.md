# C. 용어집

## Archive Agent

Daily Note나 inbox note를 읽고 curated note, project note, question queue, MOC, graph report로 정리하는 Agent.

## Capture Protocol

사용자가 기록할 때 지키는 최소 구조. 이 책에서는 Daily Note의 `Insight`, `Learn`, `Project`, `Question`, `Decision`, `Meeting`, `Reference` heading을 기본 protocol로 삼는다.

## MOC

Map of Content. 폴더 목록이 아니라 특정 주제를 탐색하기 위한 curated navigation note.

## Orphan Note

들어오거나 나가는 링크가 거의 없어 graph에서 고립된 노트.

## Hub Note

많은 노트가 연결되는 중심 노트. 좋은 navigation entry일 수도 있고, 너무 일반적인 링크가 몰린 노이즈일 수도 있다.

## Provenance

노트의 출처와 생성 근거. Agent가 만든 노트는 원본 Daily Note, reference URL, ingest source를 추적 가능하게 남겨야 한다.
